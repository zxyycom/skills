# Decision Records

`decision-records` 用稳定 Markdown 文件保存重要长期判断的完整目的、背景、决策和按需声明的演进关系，用版本化 JSON 索引保存当前仍作为后续工作依据的精简摘要。它关注“希望长期达成什么、为何采用当前方向、以后怎样不偏离”，不承接通用知识、任务日志或当前实现状态。

## 核心体验

日常工作继续以原任务为主。当用户正在形成可能持续影响后续行为、owner、边界、兼容性、风险处理或验收方式的决定，或拟议决定可能与既有决定冲突时，agent 会在确定方向或实施依赖该方向的实质改动前读取 `docs/decisions/decision-index.json`，通过路径、标题和摘要中的目的、背景、决策恢复相关当前判断，再结合完整记录、当前事实和行为 owner 判断是否沿用。

发现当前任务与既有判断不一致时，agent 会区分明确的一次性例外、需要确认的长期修订和记录或 owner 之间的一致性问题。只改变本次任务执行方式的指令属于一次性例外；只有用户已经决定改变后续工作的默认依据时才属于长期修订。

当前指令已经表达决策维护意图时，agent 会直接执行而不重复询问；新增、调整或退出当前集合前，会明确告知用户将写入或改变的判断及集合变化。

后续维护先从 JSON 索引读取当前判断，需要审计演进时再沿 Markdown 直接关系查看历史。不在索引中的记录属于逻辑归档，但保留原路径并可通过 CLI 查询；没有后续判断的普通退役不强制建立关系。

已有材料不符合固定契约时，skill 先报告差异、可保留语义和修复范围。格式迁移需要独立确认，完成后只维护一种格式。

## 设计边界

1. 可审核：JSON 当前成员、Markdown 内容和历史关系都能进入版本审查。
2. 可演进：后续判断通过当前成员切换和单向直接关系表达演进；没有后续判断的普通退役可以独立退出当前集合，不覆盖形成原因。
3. 可定位：索引只包含当前判断，并直接提供路径、标题、精简目的、背景和决策；摘要由 Markdown 显式提供，主题分类由路径表达。
4. 语义分层：完整章节承接完整语义，Markdown 摘要只压缩对应章节，JSON 以路径承接当前成员并镜像这些摘要。
5. 路径稳定：文件创建后不因生命周期、关系或标题润色而重命名或移动。
6. 防偏离：长期决策形成或与既有决定冲突时恢复相关判断，偏离必须归类，不把一次性例外静默沉淀成新方向。
7. 逻辑归档：历史记录退出索引但留在原主题分类，通过 `--archived` 或 `--all` 按需查询。

## Owner 分工

1. [`SKILL.md`](../../skills/decision-records/SKILL.md) 承接 agent 的发现、确认、读取、写入和交付流程。
2. [`decision-record-rules.md`](../../skills/decision-records/references/decision-record-rules.md) 是目录、索引、稳定身份、正文、关系和维护事务的唯一固定契约。
3. `scripts/decision-records/` 承接 CLI 源码、测试和构建；随 skill 分发的 CLI 提供结构检查、当前与归档查询、关系链反查、派生字段同步和校验式成员更新。
4. 每条 Markdown 的完整章节承接完整语义，同文件的索引摘要承接作者确认的压缩表达；`docs/decisions/decision-index.json` 只承接当前成员并镜像摘要派生字段。

## CLI 体验

1. `check` 检查 JSON schema、当前成员、稳定文件名、正文结构和直接关系。
2. `list` 默认展示当前记录，`--archived` 展示逻辑归档，`--all` 展示完整历史。
3. `trace <path>` 从任一当前或归档记录出发，默认查询完整前后链；`--direction predecessors|successors|both` 选择方向，`--depth <n>` 限制关系跳数。
4. `sync-index --write` 只刷新标题、精简目的、背景、决策和排序，不自行判断当前成员。
5. `activate` 将独立新判断加入当前集合。
6. `archive <old...>` 独立归档不再使用的判断；`archive <old...> --by <new>` 校验后续记录直接引用全部前序，再归档前序并激活后续。校验失败时恢复原索引。

## 后续方向

后续通过真实样例继续校准主动恢复范围、偏离分类、候选概览信息量和非标准格式修复报告。目标项目若需要冷启动路由，应在自己的 agent 指令中只加入指向 JSON 当前索引和本 skill 的短入口，不复制契约，也不增加新的记忆系统。
